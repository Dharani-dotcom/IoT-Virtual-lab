# 🌐 Virtual IoT Sensor Laboratory

An interactive IoT Virtual Lab built with Streamlit.

## Features

- Temperature Sensor Simulation
- Soil Moisture Sensor
- Humidity Sensor
- Gas Detection
- Light Sensor
- Motion Detection

## Technologies Used

- Python
- Streamlit
- Pandas
- NumPy

## Run the Project

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## Purpose

This project helps ECE students experiment with IoT sensors without physical hardware.
